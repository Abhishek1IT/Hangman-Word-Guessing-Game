# Hangman Word Guessing Game

A simple and interactive Hangman Word Guessing Game built using **HTML**, **CSS**, and **Vanilla JavaScript** without using any external libraries or frameworks.

## Features

* Random word selection
* Hint system for each word
* Dynamic word display using underscores
* One-by-one letter reveal
* Attempts counter
* Win and lose conditions
* Dynamic game logic
* Input validation
* Responsive user interface
* Pure Vanilla JavaScript implementation

1. Download or clone the repository
2. Open the project folder

---

## Game Rules

* A random word is selected by the computer.
* The player guesses one letter at a time.
* Correct letters are revealed in the word.
* Each guess decreases the remaining attempts.
* The player wins when the required letters are revealed.
* The player loses when all attempts are finished.
